# 📝 Notes App

A simple Notes Application built using **HTML, CSS, and JavaScript** with **Local Storage** for data persistence. This project includes user authentication (Signup/Login) and note management features without using any backend or database.

## 🚀 Features

### Authentication

* User Signup
* User Login
* Session Management using Local Storage
* Logout Functionality

### Notes Management

* Create Notes
* View Notes
* Delete Notes
* Search Notes
* User-specific Notes Storage

### Storage

* Uses Browser Local Storage
* No Database Required
* No Backend Required
